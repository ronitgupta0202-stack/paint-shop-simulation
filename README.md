# Paint Shop Simulation

## Overview

This project simulates the workflow of an automobile paint shop using Python. It models different processing stations such as Cleaning, Primer, and Painting while tracking queue lengths, waiting times, resource utilization, and production performance.

## Features

* Event-driven paint shop simulation
* Queue and waiting time analysis
* Machine utilization calculation
* Performance statistics
* Alert generation for high queue conditions

## Technologies Used

* Python 3

## How to Run

```bash
py main.py
```

## Output

The simulation displays:

* Station utilization
* Maximum queue length
* Average waiting time
* Number of alerts triggered
